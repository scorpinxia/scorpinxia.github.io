联机算法

很容易看出，联机装箱不存在最优，因为不知道何时结束。，前面的M个小项可以装到 M/2个箱子里，但是后面的M个大箱，都要占一个箱子

任意联机算法至少使用4/3最优箱子

下面是三种简单算法保证箱子不多于两倍的最优

  • 下项适合算法 （next fit)
    • 处理任何物品，检查能否装进刚刚物品的同一个箱子
    • 绝不超过两倍最优，但是有许多达到差不多2倍
    • O(N)完成
  • 首项适合算法（first fit)
    • 扫描之前的箱子有没有可以放入的地方
    • ，大量均匀分布的时候，first fit 比最优多20%，完全可以接受
    • 一般是O(N^2) 可以是O(NlogN)
  • 最佳适合算法 （best fit）
    • 不是找到第一个可以放的，而是找最满的存放位置
    • 由于一般的坏情况是一样的，所以不会有什么改善
    • 一般是O(N^2) 可以是O(NlogN)

脱机算法

将各项物品排序，把最大的放在最前面，然后在用首次适合或者最近适合算法

  • 首次适合递减算法（first fit decreasing）

    • 所有权重大于1/3的项放入前M个，其余物品的项数最多为M-1(前面M个箱子每箱最多两个，一定小于M-1物品，不然就被放进去了）

    • M个最优，首次适合不超过 （4M+1)/3

    • 均匀输入 期望θ(根号M)

分治算法(divide and conquer)

  • 分：递归解决较小的问题
  • 治：从子问题的解构建原问题的解

传统上至少两个递归调用才叫分治算法

最大子序和O(NlogN) 、树遍历、归并排序和快速排序

运行时间

T(N)=2T(N/2)+O(N) 每个问题是原问题的一半加上O(N)的附加工作 解为O(NlogN)

最近点问题

找一对距离最近的点（x1,y1) (x2,y2)

  • 一般方法 算N(N-1)个距离
  • 先按x排序O(NlogN), 把点集分一半 Pl、Pr 那么最小距离只有Dl、Dr、Dc两个点集的最小

计算dc

  • 蛮力，计算在最小距离的中心带

  • 计算 y之前的距离

问题是 y坐标还需要排序, O(Nlog^2N),可以保留两个表，按照x排序和y排序的 P Q 每次传入的是Pl Ql

选择问题

找第k个最小元素

  • 五数中值法 选枢纽元
    • N个元素分成 N/5组 5个元素的组，忽略剩余元素
    • 找每组的中值 N/5个中值
    • 找N/5个中值的中值 作为枢纽元
  • 这样的快速选择方法运行时间为O(N)

public class Problem10_2 {

    String getMostBenefit(HomeWork[] h){
        StringBuffer s=new StringBuffer();
        Arrays.sort(h);
        int maxEndTime=0;
        for (int i = 0; i <h.length ; i++) {
            if(h[i].endTime>maxEndTime){
                maxEndTime=h[i].endTime;
            }

        }

        int[] arrTime=new int[maxEndTime];
        for (int i = h.length-1; i >=0 ; i--) {
            for (int j = h[i].endTime-1; j >=0 ; j--) {
                if(arrTime[j]==0){
                    arrTime[j]=i;
                    //s.append(h[i]);
                    break;
                }
            }
        }

        for (int i = 0; i < arrTime.length; i++) {
            s.append(h[arrTime[i]]);
        }
        return s.toString();
    }

    public static void main(String[] args) {

        HomeWork[] h=new HomeWork[5];
        h[0]=new HomeWork("A",2,100);
        h[1]=new HomeWork("B",1,19);
        h[2]=new HomeWork("C",2,27);
        h[3]=new HomeWork("D",1,25);
        h[4]=new HomeWork("E",3,15);


        Problem10_2 p=new Problem10_2();
        System.out.println(p.getMostBenefit(h));
    }
}

10.6 huffman算法在有序时 只需要O(N)时间

维护两个队列Q1和Q2。 Q1将按排序顺序存储单节点树，而Q2将按排序顺序存储多节点树。将初始的单节点树放在Q1上，首先使最小的权重树排队。最初，Q2为空。检查Q1和Q2的每一个的前两个条目，并使两个最小的条目出队。 （这需要轻松实现ADT的扩展。）合并树并将结果放在Q2的末尾。继续此步骤，直到Q1为空，并且Q2中只剩下一棵树。

    private double getMinDistanceByXY(Point[] pArrX,Point[] pArrY){
        return getMinDistanceByXY(pArrX,0,pArrX.length-1);
    }

    private double getMinDistanceByXY(Point[] pArrX, int left, int right) {
        
        if(left==right){
            return Double.MAX_VALUE;
        }

        if(left==right-1){
            return distance(pArrX[left],pArrX[right]);
        }
        int center=(right+left)/2;

        double dLeftMin=getMinDistanceByXY(pArrX,left,center);
        double dRightMin=getMinDistanceByXY(pArrX,center+1,right);


        double dCenterMin=Math.min(dLeftMin,dRightMin);
        double centerX=(pArrX[center].x+pArrX[center+1].x)/2;

        double t;

        int leftInCenter,rightInCenter;

        //下标遍历到左边第二个即可，第一个没意义，还可能导致leftInCenter=-1
        for (leftInCenter = center; leftInCenter >0 ; leftInCenter--) {
            if (centerX - pArrX[leftInCenter].x > dCenterMin) {
                break;
            }
        }


        for (rightInCenter = center+1; rightInCenter <right ; rightInCenter++) {
            if (pArrX[rightInCenter].x - centerX > dCenterMin) {
                break;
            }
        }



        Arrays.sort(pArrX,leftInCenter,center,new CmpPointY());
        Arrays.sort(pArrX,center,rightInCenter, Collections.reverseOrder(new CmpPointY()));


        for (int i = center; i >=leftInCenter ; i--) {
            for (int j = center+1; j <=right ; j++) {
                if(pArrX[i].y-pArrX[j].y>dCenterMin){
                    break;
                }
                t=distance(pArrX[i],pArrX[j]);
                if(t<dCenterMin){
                    dCenterMin=t;
                }

            }
        }
        return dCenterMin;
        
    }

    private double getMinDistanceByX(Point[] pArrX){
        return getMinDistanceByX(pArrX,0,pArrX.length-1);
    }

    private double getMinDistanceByX(Point[] pArrX, int left, int right) {

        if(left==right){
            return Double.MAX_VALUE;
        }

        if(left==right-1){
            return distance(pArrX[left],pArrX[right]);
        }

        double dLeftMin=getMinDistanceByX(pArrX,left,(right+left)/2);
        double dRightMin=getMinDistanceByX(pArrX,(right+left)/2+1,right);


        double dCenterMin=Math.min(dLeftMin,dRightMin);
        double centerX=(pArrX[(left+right)/2].x+pArrX[(left+right)/2+1].x)/2;

        double t;
        for (int i = (right+left)/2; i >=left ; i--) {
            if(centerX-pArrX[i].x>dCenterMin){
                break;
            }
            for (int j = (right+left)/2+1; j <=right ; j++) {
                if(pArrX[j].x-centerX>dCenterMin){
                    break;
                }
                t=distance(pArrX[i],pArrX[j]);
                if(t<dCenterMin){
                    dCenterMin=t;
                }

            }
        }

        return dCenterMin;

    }

    private double getMinDistance(Point[] pArrX){
        double minD=Double.MAX_VALUE;
        double t;
        for (int i = 0; i <pArrX.length ; i++) {
            for (int j = i+1; j < pArrX.length; j++) {
                t=distance(pArrX[i],pArrX[j]);
                if(t<minD)
                    minD=t;
            }
        }

        return minD;
    }

    private static double distance(Point pArrX, Point pArrX1) {
        return Math.sqrt(Math.pow(pArrX.x-pArrX1.x,2)+Math.pow(pArrX.y-pArrX1.y,2));
    }
}

    }

}

class CmpPointY implements Comparator<Point>{
    @Override
    public int compare(Point p1, Point p2) {
        return Double.compare(p1.y,p2.y);
    }

}

    public static final long INFINITY = Long.MAX_VALUE;

    /**
     * Compute optimal ordering of matrix multiplication.
     * c contains the number of columns for each of the n matrices.
     * c[ 0 ] is the number of rows in matrix 1.
     * The minimum number of multiplications is left in m[ 1 ][ n ].
     * Actual ordering is computed via another procedure using lastChange.
     * m and lastChange are indexed starting at 1, instead of 0.
     * Note: Entries below main diagonals of m and lastChange
     * are meaningless and uninitialized.
     */
    public static void optMatrix( int [ ] c, long [ ][ ] m, int [ ][ ] lastChange )
    {
        int n = c.length - 1;

        for( int left = 1; left <= n; left++ )
            m[ left ][ left ] = 0;
        for( int k = 1; k < n; k++ )   // k is right - left
            for( int left = 1; left <= n - k; left++ )
            {
                // For each position
                int right = left + k;
                m[ left ][ right ] = INFINITY;
                for( int i = left; i < right; i++ )
                {
                    long thisCost = m[ left ][  i ] + m[ i + 1 ][ right ]
                            + c[ left - 1 ] * c[ i ] * c[ right ];
                    if( thisCost < m[ left ][ right ] )  // Update min
                    {
                        m[ left ][ right ] = thisCost;
                        lastChange[ left ][ right ] = i;
                    }
                }
            }
    }

    int minMutiply(int[] C){
        return minMutiply(C,1,C.length-1);
    }

    int minMutiply(int[] C,int left,int right){
        if(left>=right){
            return 0;
        }

        if(left==right-1){
            return C[left-1]*C[left]*C[right];
        }


        int minM=Integer.MAX_VALUE;
        int tmp;
        for (int i = left; i <right ; i++) {
            tmp=minMutiply(C,left,i)+minMutiply(C,i+1,right)+C[left-1]*C[i]*C[right];
            if(tmp<minM){
                minM=tmp;
            }
        }
        return minM;
    }
}

        float nums[]={0.22f,0.18f,0.20f,0.05f,0.25f,0.02f,0.08f};
        int[][] lastChange=new int[nums.length][nums.length];
        float[][] m=new float[nums.length][nums.length];


        minSearchTree(nums,m,lastChange);
        for( int i = 0; i < nums.length; i++ )
        {
            for( int j = 0; j < nums.length; j++ )
                System.out.print( m[ i ][ j ] + "    " );
            System.out.println( );
        }
        for( int i = 0; i < nums.length; i++ )
        {
            for( int j = 0; j < nums.length; j++ )
                System.out.print( lastChange[ i ][ j ] + "    " );
            System.out.println( );
        }

    }

    public static final float INFINITY = Float.MAX_VALUE;
    private static void minSearchTree(float[] c,float[][] m,int[][] lastChange){
        int n=c.length;

        for (int i = 0; i < n; i++) {
            m[i][i]=c[i];
        }

        //k=right-left
        for (int k = 1; k < n ; k++) {
            for (int left = 0; left <n-k ; left++) {

                int right=left+k;
                m[left][right]=INFINITY;
                for (int i = left; i <= right; i++) {
                    float tmp=getM(m,left,i-1)+getM(m,i+1,right)+sum(c,left,right);

                    if(tmp<m[left][right]){
                        lastChange[left][right]=i;
                        m[left][right]=tmp;
                    }
                }

            }
        }
    }

    private static float getM(float[][] m, int left, int right) {
        if(left>right)
            return 0;

        return m[left][right];
    }


    private static float sum(float[] c, int left, int right) {
        float sum=0;
        for (int i = left; i <=right; i++) {
            sum+=c[i];
        }

        return sum;
    }
}

    private static boolean turnPike(int[] x, List<Integer> list, int n) {
        x[1]=0;
        x[n]=deleteMax(list);
        x[n-1]=deleteMax(list);

        System.out.println(list);

        if(list.contains(x[n]-x[n-1])){
            remove(list,x[n]-x[n-1]);
            System.out.println(list);
            return place(x,list,n,2,n-2);
        }

        return true;
    }

    private static boolean place(int[] x, List<Integer> distance, int n, int left, int right) {
        int dmax;
        boolean found=false;

        if(distance.isEmpty()) {
            return true;
        }

        dmax=findMax(distance);

        if(isInDistanceForAll(x,distance,dmax,left,right)){
            x[right]=dmax;
            for (int j = 1; j <left; j++) {
                distance.remove((Integer)Math.abs(x[j]-dmax));
            }

            for (int j = right+1; j <x.length; j++) {
                distance.remove((Integer)Math.abs(x[j]-dmax));
            }
            found=place(x,distance,n,left,right-1);

            if(!found){
                for (int j = 1; j <left; j++) {
                    distance.add(Math.abs(x[j]-dmax));
                }

                for (int j = right+1; j <x.length; j++) {
                    distance.add(Math.abs(x[j]-dmax));
                }
            }else {
                System.out.println(distance);
            }



        }


        if(!found && isInDistanceForAll(x,distance,x[n]-dmax,left,right)){
            x[left]=x[n]-dmax;
            for (int j = 1; j <left; j++) {
                distance.remove((Integer) Math.abs(x[n]-dmax-x[j]));
            }

            for (int j = right+1; j <x.length; j++) {
                distance.remove((Integer) Math.abs(x[n]-dmax-x[j]));

            }
            found=place(x,distance,n,left+1,right);

            if(!found){
                for (int j = 1; j <left; j++) {
                    distance.add(Math.abs(x[n]-x[j]-dmax));
                }

                for (int j = right+1; j <x.length; j++) {
                    distance.add(Math.abs(x[n]-x[j]-dmax));
                }
            }else {
                System.out.println(distance);
            }
        }


        return found;
    }

    private static boolean isInDistanceForAll(int[] x,List<Integer> distance,int dmax, int left, int right) {
        boolean flag=true;
        int leftHigh=left;
        int rightHigh=x.length;
        for (int j = 1; j <left; j++) {
            if(!distance.remove((Integer)Math.abs(x[j]-dmax))){
                flag = false;
                leftHigh=j;
                break;
            }
        }

        for (int j = right+1; j <x.length; j++) {
            if(!distance.remove((Integer)Math.abs(x[j]-dmax))){
                flag = false;
                rightHigh=j;
                break;
            }
        }


        for (int j = 1; j <leftHigh; j++) {
            distance.add(Math.abs(x[j]-dmax));
        }

        for (int j = right+1; j <rightHigh; j++) {
            distance.add(Math.abs(x[j]-dmax));
        }
        return flag;
    }

    private static int findMax(List<Integer> list) {
        int max=0;
        int maxindex=-1;
        for (int i = 0; i < list.size(); i++) {

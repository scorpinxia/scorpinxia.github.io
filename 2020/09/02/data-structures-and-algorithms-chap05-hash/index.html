解决冲突

分离链接法

冲突插入的放在最前面（最可能访问）

  • 装填因子（load factor）λ 表元素个数和表大小的比。链表的平均长度也是λ

  • 所以不成功查找，需要考察平均λ个节点。成功查找需遍历1+λ/2个链表：由下说明，在N个元素的散列表以及M个链表。其他节点的概率 (N-1)/M (相当又找到这N-1个)，因为假设M比较大，平均一半其他节点被搜索到

  • 分离链接法一般法就是使 λ=1，超过的话就rehash

  • 预期探测次数 不成功查找和插入一致 1/2(1+1/(1-λ)^2) 成功 1/2(1+1/(1-λ))

标准库散列

HashSet、HashMap。HashSet 的关键字必须实现Hashcode和equals(显然)，他们通常是分离链接散列

如果是否有序无关，用这些类很好 如4.8三种映射

  • key为一个单词，关键词是与单词只有一个字母不同的所有单词。除非我们想依次打印，也可以是HashMap

最坏O(1)的散列表

合理装填因子和散列函数 插入、删除、查找都是O(1)。假设散列函数良好，最坏情形的期望。

分离链接来说，λ=1，变成球盒问题，N个球均匀随机放在N个盒子，装球最多盒子球的期望 θ（LogN/loglogN)、探测散列最长期望类似

事先知道N的值在插入过程重新排列，得到最差O（1）的情形

完美散列(所有项优先知道)

N个球放入M=N^2个盒子，没任何盒子装超过一个球的概率不小于1/2

证明：一共有N(N-1)/2对冲突，冲突最多N(N-1)/2对，冲突概率是1/M 所以最大为N(N-1)/2N^2=1/2

但是不能那么大的M，只用N个盒子，用散列表解决盒子冲突，根据每个位置的项数决定二级散列表

N个球放入M=N个盒子的主散列表，二级散列表容量期望最多为2N

证明：冲突最多期望N(N-1)/2N,加上项数总个数N，所以总的二级空间需求量超过4N概率最多是1/2

布谷鸟散列 （允许动态插入删除，选择的力量）

科学家证明：如果随机选择两个盒子，投入比较空的那个，最大盒子容量仅仅是θ（loglogN）

一次查找最多访问两次表，找到删除也简单（不需懒惰删除）

阻碍插入循环的概率多大，λ《0.5是很低，若干次替换检测到可以重新用心的散列。

实现

重点理解插入

  • 基本计划：存在返回，满载扩展，正常情况的插入
  • 正常情况的插入：rehashs跟踪再散列次数，基本逻辑和经典方法不一样，检测任何有效位置是否是空的，放在第一个空的位置，不然就替换其中一个已经存在的。
    • 出现的问题：替换第一项实验表现不好
    • 替换最后一项实验表现不好
    • 按序列替换（第一次hash0,第二次hash1……）表现不好
    • 随机替换表现不好，特别是两个散列时容易循环

新的重新随机替换方法，不试图替换最后一项，太多替换就尝试新的散列函数，太多再散列就扩展表。

   private static final double MAX_LOAD = 0.40;
   private static final int ALLOWED_REHASHES = 1;
   
   private int rehashes = 0;
     
   private boolean insertHelper1( AnyType x )
   {
       final int COUNT_LIMIT = 100;
       
       while( true )
       {
           int lastPos = -1;
           int pos;

           for( int count = 0; count < COUNT_LIMIT; count++ )
           {
               for( int i = 0; i < numHashFunctions; i++ )
               {
                   pos = myhash( x, i );

                   if( array[ pos ] == null )
                   {
                       array[ pos ] = x;
                       currentSize++;
                       return true;
                   }
               }

               // none of the spots are available. Kick out a random one
               int i = 0;
               do
               {
                   pos = myhash( x, r.nextInt( numHashFunctions ) );
               } while( pos == lastPos && i++ < 5 );

               AnyType tmp = array[ lastPos = pos ];
               array[ pos ] = x;
               x = tmp;
           }

           if( ++rehashes > ALLOWED_REHASHES )
           {
               expand( );      // Make the table bigger
               rehashes = 0;
           }
           else
               rehash( );
       }
   }

跳房子散列（改进经典线性探测)

现代体系中探测相邻单元产生的位置和额外探测比，前者更重要，线性探测仍然重要，甚至是最佳

跳房子散列：用事先确定对底层体系最优常数。给探测序列最大长度加上界，给出常数最坏，和布谷鸟一样查询并行化。

梅森素数（Mersenne）

由上可以选择任何计算有利的素数 P=2^31-1 2^5-1 2^61-1.乘以这样的p可以用位运算和减法完成。取模运算也可以用位移运算和加法完成

mod 法则

  • 结合律
    • ((a+b) mod p + c)mod p = (a + (b+c) mod p) mod p
    • ((a×b) mod p × c)mod p = (a * (b×c) mod p) mod p
  • 交换律
    • (a + b) mod p = (b+a) mod p
    • (a × b) mod p = (b × a) mod p
  • 分配律
    • ((a +b)mod p × c) mod p = ((a × c) mod p + (b × c) mod p) mod p
    • (a×b) mod c=(a mod c × b mod c) mod c
    • (a+b) mod c=(a mod c+ b mod c) mod c
    • (a-b) mod c=(a mod c- b mod c) mod c

卡特-韦格曼Trick。r=y(mod p)。 y除以(p+1),y=q’(p+1)+r’,两边同时取mod 所以r=q’+r’(mod p)

可扩散列

数据不再主存里面，任意时刻有N个记录存储。 最多可以把M个记录放在一个磁盘，对于前面的散列需要O(N)次磁盘

可扩散列，可以用两次磁盘访问执行，如下D是bit数，按照第四章的B树，每个树叶最多M个袁术，数据前2bit相等

性能： 目录大小期望O(N^(1+1/M)/M) M太小 主存装不下 就需要第二次磁盘

  • 游戏程序转移表

public class Problem5_13 {

    List mutilpy_Ployn(LinkedList<Items> l1, LinkedList<Items> l2) {
        LinkedList<Items> l3 = new LinkedList<>();

        for (Items i : l1) {
            for (Items j : l2) {
                Items tmp = new Items();
                tmp.a = i.a * j.a;
                tmp.n = i.n + j.n;
                insert(l3, tmp.a, tmp.n);
            }

        }

        return l3;
    }

    private void insert(LinkedList<Items> l3, double a, int n) {
        ListIterator<Items> l3Iterator = l3.listIterator();
        Items tmp = new Items(a, n);


        Items cur;
        while (l3Iterator.hasNext() && tmp.n <= (cur = l3Iterator.next()).n) {

            if (cur.n == tmp.n) {
                cur.a += tmp.a;
                if (cur.a == 0) {
                    l3Iterator.remove();
                }
                return;
            }
        }
        if (l3Iterator.hasNext()) {
            l3Iterator.previous();
            l3Iterator.add(tmp);
        } else {
            l3Iterator.add(tmp);
        }

    }


    public static void main(String[] args) {
        Problem5_13 p = new Problem5_13();
        LinkedList<Items> l1 = new LinkedList<>();
        p.insert(l1, 3, 1);
        p.insert(l1, 3, 0);

        LinkedList<Items> l2 = new LinkedList<>();
        p.insert(l2, 3, 1);
        p.insert(l2, 1, 0);

        List l3 = p.mutilpy_Ployn(l1, l2);
    }

    class Items {
        double a;
        int n;

        Items() {
            this(0, 0);
        }

            this.a = a;

        }

    • 插入策略，重复插入一定次数后选择新的散列函数
      • 当全部散列函数满了,选择哪个替代（随机法）
      • 每次都依次替代（非满时也是，会容易造成表很大，因为任意rehash和expand

package chap05;

import java.util.Random;

/**
 * @ClassName MyCuckooHashTable
 * @Author: Wibur
 * @Description:
 * @Date: Created in 9:28 2020/9/12
 * @Versuin: 1.0
 */
public class MyCuckooHashTable <AnyType> {

    private static int DEFUALT_SZIE = 101;
    private double MAX_LOAD = 0.40;
    private int COUNT_LIMIT = 100;
    private int RESH_HASH_ALLOW = 1;

    private int rehashNum = 0;
    private int currentSize = 0;

    private Random r = new Random();
    private AnyType[] arr;
    private HashFamily<? super AnyType> hashFamily;

    MyCuckooHashTable(HashFamily<? super AnyType> hf) {
        this(hf, DEFUALT_SZIE);
    }

    public MyCuckooHashTable(HashFamily<? super AnyType> hf, int defualtSzie) {
        hashFamily = hf;
        allocateArray(defualtSzie);
        makeClear();

    }

    public boolean insert(AnyType x) {
        if (contains(x))
            return false;

        if (currentSize >= arr.length * MAX_LOAD)
            expand();

        return insertHelper2(x);
    }

    private boolean insertHelper1(AnyType x) {
        int funLength = hashFamily.getNumberOfFunctions();
        int lastPos = -1;
        int pos=-1;
        while (true) {
            for (int i = 0; i < COUNT_LIMIT; i++) {
                for (int j = 0; j < funLength; j++) {
                    pos = myHash(x, j);
                    if (arr[pos] == null) {
                        arr[pos] = x;
                        currentSize++;
                        return true;
                    }
                }
                int j = 0;
                do
                {
                    lastPos = myHash( x, r.nextInt( funLength ) );
                } while( lastPos==pos && j++ < 5 );

                AnyType tmp = arr[ lastPos ];
                arr[ lastPos ] = x;
                x = tmp;
            }

            if (++rehashNum > RESH_HASH_ALLOW) {

                expand();
                rehashNum=0;
            } else
                rehash();
        }

    }

    private boolean insertHelper2( AnyType x )
    {
        final int COUNT_LIMIT = 100;

        while( true )
        {
            for( int count = 0; count < COUNT_LIMIT; count++ )
            {
                int pos = myHash( x, count % hashFamily.getNumberOfFunctions() );

                AnyType tmp = arr[ pos ];
                arr[ pos ] = x;

                if( tmp == null )
                    return true;
                else
                    x = tmp;
            }

            if( ++rehashNum > RESH_HASH_ALLOW )
            {
                expand( );      // Make the table bigger
                rehashNum = 0;
            }
            else
                rehash( );
        }
    }

    public boolean remove(AnyType x){
       int pos=findPos(x);

        if (pos==-1)
            return false;

        arr[pos]=null;
        currentSize--;
        return true;

    }


    private void rehash() {
        hashFamily.generateNewFunctions();
        rehash(arr.length);
    }

    private void expand() {

        rehash((int) (arr.length/MAX_LOAD));
    }

    private void rehash(int newLength) {
        AnyType[] oldArr=arr;
        allocateArray(newLength);

        currentSize=0;
        for (AnyType item:oldArr) {
           if(item!=null)
               insert(item);
        }
    }

    private int findPos(AnyType x){
        int pos;
        int length=hashFamily.getNumberOfFunctions();
        for (int i = 0; i < length; i++) {
            pos=myHash(x,i);
            if(arr[pos]!=null&&arr[pos].equals(x))
                return pos;
        }

        return -1;
    }

    private int myHash(AnyType x, int i) {
        int pos=hashFamily.hash(x,i);

        pos%=arr.length;

        if(pos<0)
            pos+=arr.length;

        return pos;

    }

    private boolean contains(AnyType x) {

        return findPos(x)!=-1;
    }

    private void allocateArray( int defualtSzie) {
        arr= (AnyType[]) new Object[nextPrime(defualtSzie)];


    }



    private static int nextPrime( int n )
    {
        if( n % 2 == 0 )
            n++;

        for( ; !isPrime( n ); n += 2 )
            ;

        return n;
    }


    private static boolean isPrime( int n )
    {
        if( n == 2 || n == 3 )
            return true;

        if( n == 1 || n % 2 == 0 )
            return false;

        for( int i = 3; i * i <= n; i += 2 )
            if( n % i == 0 )
                return false;

        return true;
    }

    private int capacity() {
        return arr.length;
    }

    private boolean isEmpty(){
        return currentSize==0;
    }

    private void makeClear(){
        for (int i = 0; i <arr.length ; i++) {
            arr[i]=null;
        }
        currentSize=0;
        rehashNum=0;
    }

    {

        final int NUMS = 2000000;
        final int GAP  =   37;

public class MyCukooHashTableClassic<AnyType> {

    private static final int DEFAULT_TABLE_SIZE = 101;
    private static final double MAX_LOAD = 0.49;
    private static final int ALLOWED_REHASHES = 100;

    private int rehashes = 0;



    private final HashFamily<? super AnyType> hashFunctions;
    private final int numHashFunctions;
    private AnyType [ ] array; // The array of elements
    private int currentSize;
    private int subTableSize;
    private int [ ] subTableStarts;


    public MyCukooHashTableClassic( HashFamily<? super AnyType> hf )
    {
            this( hf, DEFAULT_TABLE_SIZE );
        }

    public MyCukooHashTableClassic( HashFamily<? super AnyType> hf, int size )
        {
            hashFunctions = hf;
            numHashFunctions = hf.getNumberOfFunctions( );
            subTableSize = nextPrime( size / numHashFunctions );
            subTableStarts = new int[ numHashFunctions ];
            for( int i = 0; i < numHashFunctions; i++ )
                subTableStarts[ i ] = i * subTableSize;

            allocateArray( subTableSize * numHashFunctions );
            doClear( );
        }


        public boolean insert( AnyType x )
        {


            if( contains( x ) )
                return false;

            if( currentSize >= array.length * MAX_LOAD )
                expand( );

            return  insertHelper1(x);

        }

    private boolean insertHelper1(AnyType x) {
        final int COUNT_LIMIT = 100;

        while (true){
            for( int i = 0, which = 0; i < COUNT_LIMIT; i++, which++ )
            {
                if( which == numHashFunctions )
                    which = 0;

                int pos = myhash( x, which );

                if( array[ pos ] == null )
                {
                    array[ pos ] = x;
                    currentSize++;
                    return true;
                }

                AnyType tmp = array[ pos ];
                array[ pos ] = x;
                x = tmp;
            }

            if( rehashes++ >= ALLOWED_REHASHES )
            {
                expand( );
                rehashes = 0;
            }
            else
                rehash( );

        }

    }

    protected int myhash( AnyType x, int which )
        {
            int hashVal = hashFunctions.hash( x, which );

            hashVal %= subTableSize;
            if( hashVal < 0 )
                hashVal += subTableSize;

            return hashVal + subTableStarts[ which ];
        }

        private void expand( )
        {
            rehash( numHashFunctions * nextPrime( (int) ( array.length / MAX_LOAD  / numHashFunctions ) ) );
        }

        private void rehash( )
        {
            //System.out.println( "NEW HASH FUNCTIONS " + array.length );
            hashFunctions.generateNewFunctions( );
            rehash( array.length );
        }

        private void rehash( int newLength )
        {
            //   System.out.println( "REHASH: " + array.length + " " + newLength + " " + currentSize );

            AnyType [ ] oldArray = array;    // Create a new double-sized, empty table

            if( newLength != array.length )
            {
                subTableSize = newLength / numHashFunctions;
                for( int i = 0; i < numHashFunctions; i++ )
                    subTableStarts[ i ] = i * subTableSize;
            }

            allocateArray( newLength );

            currentSize = 0;

            // Copy table over
            for( AnyType str : oldArray )
                if( str != null )
                    insert( str );
        }

        /**
         * Gets the size of the table.
         * @return number of items in the hash table.
         */
        public int size( )
        {
            return currentSize;
        }

        /**
         * Gets the length (potential capacity) of the table.
         * @return length of the internal array in the hash table.
         */
        public int capacity( )
        {
            return array.length;
        }

        /**
         * Method that searches two places.
         * @param x the item to search for.
         * @return the position where the search terminates, or -1 if not found.
         */
        private int findPos( AnyType x )
        {
            for( int i = 0; i < numHashFunctions; i++ )
            {
                int pos = myhash( x, i );
                if( array[ pos ] != null && array[ pos ].equals( x ) )
                    return pos;
            }

            return -1;
        }

        /**
         * Remove from the hash table.
         * @param x the item to remove.
         * @return true if item was found and removed
         */
        public boolean remove( AnyType x )
        {
            int pos = findPos( x );

            if( pos != -1 )
            {
                array[ pos ] = null;
                currentSize--;
            }

            return pos != -1;
        }

        /**
         * Find an item in the hash table.
         * @param x the item to search for.
         * @return the matching item.
         */
        public boolean contains( AnyType x )
        {
            return findPos( x ) != -1;
        }

        /**
         * Make the hash table logically empty.
         */
        public void makeEmpty( )
        {
            doClear( );
        }

        private void doClear( )
        {
            currentSize = 0;
            for( int i = 0; i < array.length; i++ )
                array[ i ] = null;
        }




        /**
         * Internal method to allocate array.
         * @param arraySize the size of the array.
         */
        private void allocateArray( int arraySize )
        {
            array = (AnyType[]) new Object[ arraySize ];
        }

        /**
         * Internal method to find a prime number at least as large as n.
         * @param n the starting number (must be positive).
         * @return a prime number larger than or equal to n.
         */
        private static int nextPrime( int n )
        {
            if( n % 2 == 0 )
                n++;

            for( ; !isPrime( n ); n += 2 )
                ;

            return n;
        }

        /**
         * Internal method to test if a number is prime.
         * Not an efficient algorithm.
         * @param n the number to test.
         * @return the result of the test.
         */
        private static boolean isPrime( int n )
        {
            if( n == 2 || n == 3 )
                return true;

            if( n == 1 || n % 2 == 0 )
                return false;

            for( int i = 3; i * i <= n; i += 2 )
                if( n % i == 0 )
                    return false;

            return true;
        }

递归准则

当编写递归例程时，关键是要牢记递归的四条基本法则：

  • 1.基准情形。必须总要有某些基准情形，它无需递归就能解出。
  • 2.不断推进。对于那些需要递归求解的情形，每一次递归调用都必须要使状况朝向一种基
    准情形推进冂
  • 3.设计法则。假设所有的递归调用都能运行。
  • 4.合成效益法则（compoundinterestrule)。在求解一个问题的同一实例时，切勿在不同的递归调用中做重复性的工作。

第四条法则（连同它的名称一起）将在后面的章节证明是合理的。使用递归计算诸如斐波那契数之类简单数学数的值的想法一般来说不是一个好主意，其道理正是根据第四条法则。
只要在头脑中记住这些法则，递归程序设计就应该是简单明了的。

上界通配符（子类型通配符）

首先理解车的例子：单车是车的子类，单车有车的特性可以代表车，但是车不能代表单车。即类型向上兼容向下不兼容。

<? extends ClassType> 该通配符为 ClassType 的所有子类型。
<? extends C>
add():编译器只知道类型是 C 或 C 的子类，所以有可能是 C D E F 其中一个类型，为保证类型安全不能添加除了 null 以外的任何元素，即使是 C 本身也不行(车不能代表单车)。
get():既然编译器不知道此时集合中的元素是 C D E F 的哪一个，返回类型只能是他们共同父类 C 类型。

Collection<？ extend Shape> arr
T ISA Shape (继承或者实现类或方法)

下界通配符


 <? super C>
add():编译器只知道类型是 C 或者 C 的父类，所以有可能是 A B C 其中一个类型。编译器知道下界是 C ，根据类型向上兼容所以可以添加的元素是 C 以及 C 的子类(单车可以代表车)。
get():既然编译器不确定集合类型是 A B C 的哪一种，返回类型只能是他们的共同父类 Object 。
特点：<? extends C> 的 add() 被限制，<? super C> 的 get() 被限制

    public static int findMax1NumberK(int[] array, int k){
        int t;
        for (int i = 0; i < array.length; i++) {
            for (int j = 1; j < array.length-i; j++) {
                if(array[j]>array[j-1]){
                    t=array[j];
                    array[j]=array[j-1];
                    array[j-1]=t;
                }

            }
        }
        return array[k-1];
    }
    public static int findMax2NumberK(int[] array, int k){
        int t;

        for (int i = 0; i < k; i++) {
            for (int j = 1; j < k-i; j++) {
                if(array[j]>array[j-1]){
                    t=array[j];
                    array[j]=array[j-1];
                    array[j-1]=t;
                }

            }
        }

        for (int i = k; i <array.length ; i++) {
            if(array[i]<array[k-1]){
                continue;
            }
            for (int j = 0; j <k ; j++) {
                if(array[i]>array[j]){
                    for (int l = k-1; l > j; l--) {
                        array[l-1]=array[l];
                    }
                    array[j]=array[i];
                    break;
                }
            }

        }

        return array[k-1];
    }

    public static int[] generateRndomK(int n){
        int[] t=new int[n];
        for (int i = 0; i <t.length ; i++) {
            t[i]=(int) (Math.random()*n);
        }

        return t;
    }

package chap01;

/**
 * @ClassName Problem1_6
 * @Author: Wibur
 * @Description:
 * 编写带有下列声明的例程：
 * publicvoid permute(String str) ；
 * private void permute(char[] str， int1ow， int high) ；
 * 第一个例程是个驱动程序， 它调用第二个例程并显示String str中的字符的所有排列。如果
 * str是”abc”， 那么输出的串则是abc， acb， bac， bca， cab和cba。第二个例程使用递归。
 *
 * @Date: Created in 22:13 2020/6/3
 * @Versuin: 2.0 自己理解错了递归的含义
 */
public class Problem1_6 {



    static long count = 0;
    public static void main(String[] args) {
        Problem1_6 p6 = new Problem1_6();
        p6.permute( "abcd");
        System.out.println(count);
    }

    private void permute(String str) {
        permute(str.toCharArray(), 0, str.length()-1);
    }

    private void permute(char[] str, int low, int high) {

        for (int i = low; i <= high; i++) {
            swap(str, low, i);
            if (low + 1 == high) {
                System.out.println(new String(str));
                count++;
            } else {
                permute(str, low + 1, high);
            }
            swap(str, i, low);
        }
    }

    private void swap(char[] str, int low, int high) {
        char temp = str[low];
        str[low] = str[high];
        str[high] = temp;
    }
  }

package chap01;

/**
 * @ClassName Probelm1_13
 * @Author: Wibur
 * @Description:
 * 设计一个泛型类Collection， 它存储Object对象的集合(在数组中) ， 以及该集合的当前大小。
 * 提供publi方法isEmpty、make Empty、insert、remove和is Present。方法is Present(x)
 * 当且仅当在该集合中存在(由equals定义) 等于x的一个Object时返回true。
 * @Date: Created in 15:32 2020/6/4
 * @Versuin: 1.0
 */
public class Probelm1_13 {
    public static class Collection<E>{
        private static final int BEGINSIZE=16;
        private static int MAXSIZE=16;
        Object[] content;
        private int size;

        public Collection(){
            this.content=new Object[BEGINSIZE];
            size=0;
        }

        public boolean isEmpty(){
            return size==0;
        }

        public void makeEmpty(){
            for (int i = 0; i <MAXSIZE ; i++) {
                this.content[i]=null;
            }
            size=0;
        }

        public void insert(E e){
            if(size>MAXSIZE*3/4){
                Object[] content_new=new Object[MAXSIZE*2];
                for (int i = 0; i <MAXSIZE ; i++) {
                    content_new[i]=content[i];
                }

                content=content_new;
                MAXSIZE*=2;

            }
            this.content[size]=e;
            size++;


        }

        public void insert(E e,int index){
            this.content[index]=e;
            size++;
        }

        public void remove(int index){
            if(index>size) {
                return;
            }

            this.content[index]=null;
            size--;
        }


        public boolean IsPresent(E e){

            for (int i = 0; i <MAXSIZE ; i++) {
                if (this.content[i]==null){continue;}
                if (this.content[i].equals(e)) { return true;}

            }
            return false;
        }

        @Override
        public String toString() {
            StringBuilder s=new StringBuilder();
            for (int i = 0; i <MAXSIZE ; i++) {
                if(this.content[i]!=null) {
                    s.append(this.content[i].toString() + '\n');
                }
            }
            return s.toString();

1.14泛型Comparable

package chap01;

import java.util.ArrayList;

/**
 * @ClassName Problem1_14
 * @Author: Wibur
 * @Description:
 * 设计一个泛型类OrderedCollection， 它存储Comparable的对象的集合(在数组中) ， 以及
 * 该集合的当前大小。提供public方法isEmpty、make Empty、insert、remove、find Min和
 * find Max。find Min和find Max分别返回该集合中最小的和最大的Comparable对象的引用
 * (如果该集合为空， 则返回null) 。
 * @Date: Created in 16:33 2020/6/4
 * @Versuin: 1.0
 */
public class Problem1_14 {
    public static class OrderedCollection<E extends Comparable<? super E>>{
        private static final int BEGINSIZE=16;
        private  int MAXSIZE=16;
        Object[] content;
        private int size;

        public int getSize() {
            return size;
        }

        public  int getMAXSIZE() {
            return MAXSIZE;
        }

        public OrderedCollection(){
            this.content=new Object[BEGINSIZE];
            size=0;
        }
        public E get(int index){
            return (E) this.content[index];
        }

        public boolean isEmpty(){
            return size==0;
        }

        public void makeEmpty(){
            for (int i = 0; i <MAXSIZE ; i++) {
                this.content[i]=null;
            }
            size=0;
        }

        public void insert(E e){
            if(size>MAXSIZE*3/4){
                Object[] content_new=new Object[MAXSIZE*2];
                for (int i = 0; i <MAXSIZE ; i++) {
                    content_new[i]=content[i];
                }

                content=content_new;
                MAXSIZE*=2;

            }
            this.content[size]=e;
            size++;


        }

        public void insert(E e,int index){
            this.content[index]=e;
            size++;
        }

        public void remove(int index){
            if(index>size) {
                return;
            }

            this.content[index]=null;
            size--;
        }


        public boolean IsPresent(E e){

            for (int i = 0; i <MAXSIZE ; i++) {
                if (this.content[i]==null){continue;}
                if (this.content[i].equals(e)) { return true;}

            }
            return false;
        }

        public E Findmax(){

            int maxindex=0;

            for (int i = 0; i <this.getMAXSIZE() ; i++) {
                if(this.content[i]!=null) {
                    maxindex=i;
                    break;
                }
            }

            if (maxindex==0){return null;}
            for (int i = maxindex; i <this.getMAXSIZE() ; i++) {
                if(this.content[i]!=null) {
                    if(this.get(i).compareTo(this.get(maxindex))<0){
                        maxindex=i;
                    }
                }
            }
            return this.get(maxindex);
        }

        @Override
        public String toString() {
            StringBuilder s=new StringBuilder();
            for (int i = 0; i <MAXSIZE ; i++) {
                if(this.content[i]!=null) {
                    s.append(this.content[i].toString() + '\n');
                }
            }
            return s.toString();
        }


    }

    public static <E extends Comparable<? super E>> E FindMax(OrderedCollection<E> e){
        int maxindex=0;

        for (int i = 0; i <e.getMAXSIZE() ; i++) {
            if(e.content[i]!=null) {
                maxindex=i;
                break;
            }
        }

        if (maxindex==0){return null;}
        for (int i = maxindex; i <e.getMAXSIZE() ; i++) {
            if(e.content[i]!=null) {
               if(e.get(i).compareTo(e.get(maxindex))<0){
                   maxindex=i;
               }
            }
        }
        return e.get(maxindex);
    }


    public static void main(String[] args) {
        OrderedCollection<String> c=new OrderedCollection<>();
        c.insert("wrqwr");
        c.insert("asad");
        c.insert("rqwr");
        c.insert("uqsdwr");
        c.insert("wrqwr3");
        c.insert("uysdwr");
       // c.insert(3); 错误
        c.remove(0);
        System.out.println(c);

        String s=FindMax(c);
        System.out.println(s);

        String s1=c.Findmax();
        System.out.println(s1);

    }

package chap01;


import java.util.Arrays;
import java.util.Comparator;



/**
 * @ClassName Problem1_15
 * @Author: Wibur
 * @Description:
 * 定义一个Rectangle类， 该类提供getLength和getwidth方法。利用图1-18中的find Max
 * 例程编写一种main方法， 该方法创建一个Rectangle数组并首先找出依面积最大的
 * Rectangle对象， 然后找出依周长最大的Rectangle对象。
 * @Date: Created in 19:43 2020/6/4
 * @Versuin: 1.0
 */


public class Problem1_15 {
    public static class Rectangle{

        int length;
        int width;

        public int getLength() {
            return length;
        }

        public int getWidth() {
            return width;
        }

        public int area(){
            return this.length*this.width;
        }

        public int circlLength(){
            return (this.length+this.width)*2;
        }

        @Override
        public String toString() {
            return String.valueOf(this.length*this.width);
        }
    }
    public static void main(String[] args) {


        Rectangle[] r=new Rectangle[15];

        for (int i = 0; i <r.length ; i++) {
            r[i]=new Rectangle();
            r[i].length=i;
            r[i].width=i;
        }
        Comparator<Rectangle> c1 =
                (Rectangle rectangle1, Rectangle rectangle2)->{
                    return rectangle1.area()>rectangle2.area()?1:0;
                };
        Comparator<Rectangle> c2 = ( Rectangle rectangle1,Rectangle rectangle2)-> rectangle1.area()>rectangle2.area()?1:0;

        Arrays.sort(r,c1);
        for (int i = 0; i <r.length ; i++) {
            System.out.println(r[i].area());
        }

        Arrays.sort(r, (rectangle1,rectangle2)->{
            if(rectangle1.circlLength()<rectangle2.circlLength()){ return -1;}

            return rectangle1.circlLength()>rectangle2.circlLength()?1:0;
        });
        for (int i = 0; i <r.length ; i++) {
            System.out.println(r[i].area());
        }

        System.out.println("yes");
    }
